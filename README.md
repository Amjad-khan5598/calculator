# Simple Calculator

A responsive web-based calculator built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations with a clean and user-friendly interface.

##  Features

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Percentage
* Clear / Reset button

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Open the calculator in your browser
2. Click the buttons to enter numbers
3. Choose an operation (+, −, ×, ÷,%)
4. Press "=" to get the result
5. Press "C" to clear

## Project Structure

* `index.html` → Main structure
* `style.css` → Design and layout
* `script.js` → Calculator logic
